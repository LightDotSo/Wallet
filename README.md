## 🚀 Getting Started

1. Download XCode: https://developer.apple.com/xcode/
2. Install Cocoapods: https://cocoapods.org/
3. Install Pnpm: https://pnpm.io/
4. Run `pod install && pnpm install && pnpm run build`
